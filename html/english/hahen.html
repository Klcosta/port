UX/UI Design • Front-End Development • HTML • CSS • JavaScript
SousChef was a group project for the full stack coding bootcamp at the University of Pennsylvania. The project requirements were a front-end application with an external Ajax call. SousChef is an app that allows you to find new recipes and presents the recipe in an easy to follow format.

I took the role of front-end developer and designer for this project. I was responsible for the design, including logo,colors, typography and UX/UI design. From there I was responsible for the client side code. I used HTML, CSS, Bootstrap, JavaScript and JQuery.
